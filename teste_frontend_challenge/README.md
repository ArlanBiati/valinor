## 💻 Projeto

---

&nbsp;&nbsp;&nbsp;

## 📂 Como iniciar o projeto

```zsh
  # Clonar o repositório
  ❯ git clone https://github.com/ArlanBiati/valinor

  # Entrar no diretório
  ❯ cd valinor

  # Instalar dependências da aplicação
  ❯ yarn install

  # Iniciar a aplicação
  ❯ yarn start

```
